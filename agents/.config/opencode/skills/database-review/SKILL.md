## Checklist
- Migration safety, reversibility, and deployment order.
- Indexes for new access patterns.
- Constraints for data integrity.
- Transaction boundaries and race conditions.
- Backfill, locking, and large-table impact.
- Query performance and N+1 risks.
- Data retention, privacy, and audit needs.
