## Checklist
- Clear resource or action boundaries.
- Stable request and response contracts.
- Explicit validation and error formats.
- Pagination, filtering, idempotency, and retries where relevant.
- Backward compatibility and migration path.
- Auth, rate limiting, auditability, and observability.
- Tests for contract and edge cases.
